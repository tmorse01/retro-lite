# Mobile App

**Priority:** Low (Phase 5)  
**Status:** Future Enhancement

## Overview

Native mobile applications for iOS and Android.

## Features

- Native iOS/Android apps for better mobile experience.
- Offline support (sync when online).
- Push notifications for retro invites.

## Implementation Notes

- Consider React Native or Flutter for cross-platform development.
- Offline support requires local database (SQLite) and sync mechanism.
- Push notifications require device registration and notification service.

## Related Features

- All features should be mobile-friendly via responsive web first.
- Native apps can leverage existing API from [Export & Integration](./08-export-integration.md).
